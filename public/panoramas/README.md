# Location Panoramas

Place optional equirectangular 360 degree backdrop images in this folder.

Expected filenames:

- `snow-desk.jpg`
- `broad-street.jpg`
- `household.jpg`
- `registrar.jpg`
- `workhouse.jpg`
- `brewery.jpg`
- `board-room.jpg`

If a file is missing or fails to load, the app falls back to the generated prototype backdrop for that location.

Recommended format:

- 2:1 equirectangular panorama
- `4096x2048` for normal use, or `8192x4096` if the file size remains acceptable
- JPG or WebP source exported as JPG with the filenames above
- Seamless left and right edge
- Horizon near the vertical midpoint
